# 📊 采购比价助手 ComparePrice MVP

一个帮助采购人员快速完成报价录入、汇率转换、交期对比并自动评分排序的轻量级网页工具。

---

## 🚀 产品简介

ComparePrice 是一款专为采购工作场景打造的比价助手工具，支持：

- 多供应商报价录入
- 固定汇率自动转换（人民币基准）
- 价格 & 交期得分自动计算
- 综合评分排序并图表可视化
- 支持本地保存、导出 Excel、图表图片
- 本地运行，无需登录，永久免费使用

## 📌 核心功能

| 功能名称         | 说明 |
|------------------|------|
| 💰 汇率换算      | 内置固定汇率，支持 CNY、USD、EUR、JPY、HKD |
| 📈 自动评分排序  | 价格和交期自动评分，权重可自定义 |
| 📊 图表可视化    | 一键生成综合评分对比图 |
| 🧾 数据导出      | 导出为 Excel 表格 & 图表 PNG 图片 |
| ♻️ 撤销操作      | 提供单步撤销功能，避免误操作 |
| 💾 本地数据持久化 | 使用 LocalStorage 存储，不上传服务器 |
| 💡 示例数据      | 支持一键加载示例项目体验 |

---

## 🛠 使用方式

1. 打开体验链接，直接录入供应商报价信息
2. 设置权重（或使用默认值）
3. 点击“生成对比图”即可查看评分结果
4. 可导出为 Excel 或图表图片用于汇报
